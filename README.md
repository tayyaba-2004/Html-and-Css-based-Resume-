## Project Information
This project is professional **personal CV (Curriculum Vitae) webpage** created entirely using  **HTML and CSS**.
It presents my **personal information, education, skills, experience/projects, and contact details** in a **professional and responsive design**.  
Additional features include:  
- **Dark/Light Mode Toggle** for better readability.  
- **Animations** for smooth transitions and modern visual effects.  
- **Download CV** button to save the CV locally.
## Technologies Used
- **HTML5** – Structure and content of the webpage.  
- **CSS3** – Styling, animations, responsive design, and dark/light mode functionality.  
- **Responsive Design** – Ensures the CV looks professional on **desktop, tablet, and mobile devices**.
